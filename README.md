# 📈 FXナビ — 総合テクニカル判定アプリ

FX初心者向けに、**20のテクニカル指標 × 9つの時間足**で相場を総合分析し、売買判定・エントリー価格・指値(利確)・逆指値(損切り)を初心者にもわかる日本語で提示するブラウザアプリです。

サーバー不要・単一HTMLファイルで動作します。

**▶ アプリを開く: https://iwachacom-debug.github.io/fx-navi/**

## 主な機能

- **総合判定**: 「強い買い〜強い売り」の5段階+スコアメーター。判定理由を自然な日本語で説明
- **20のテクニカル指標**: 移動平均クロス・MACD・一目均衡表・パラボリックSAR・ADX/DMI・RSI・ストキャスティクス・CCI・RCI・ボリンジャーバンド・ピボット・フィボナッチ・サポレジ自動検出・ローソク足パターン など
- **9時間足のマルチタイムフレーム分析**: 1分足〜週足を加重平均。ADXでトレンド/レンジ相場を自動判別し、指標の重みを動的に切替
- **売買プラン**: エントリー価格(押し目指値の提案付き)・逆指値・指値・リスクリワード比を具体的な価格で提示
- **厳選フィルタ**: リスクリワード比不足・上位足の逆行・確信度不足・急変動直後・重要指標の接近を検出したら「見送り」と理由を表示
- **経済指標カレンダー**: 今後24時間の重要指標を日本時間・日本語名で表示。雇用統計などの発表前後は自動で見送り判定
- **市場セッション表示**: 東京/ロンドン/NY時間、早朝のスプレッド拡大警告
- **初心者向け設計**: 全用語に一言解説、用語集32項目を内蔵

## データソース

1. GMOコイン 外国為替FX Public API(APIキー不要)
2. Twelve Data(設定画面で無料APIキーを登録した場合の予備)
3. デモモード(API不通時の動作確認用・架空データ)

## 開発者向け

- `index.html` 1ファイルにすべて含まれています(Vanilla JS・ビルド不要)
- URLに `?test=1` を付けると指標計算エンジンのセルフテスト(16件)が実行されます
- 設計書: [docs/FX_SIGNAL_APP_設計書.md](docs/FX_SIGNAL_APP_設計書.md)
